---
title: "Categories API"
description: "Manage the five exercise category types: body areas, exercise types, positions, movements, and equipment types. Default categories cannot be deleted."
---

Exercise categories allow clinicians to classify exercises for filtering and discovery. There are five independent category types, each sharing the same schema and CRUD operations.

<Warning>
DELETE operations only succeed on **custom** categories (those where `is_custom` is `true`). Attempting to delete a default/system category returns `400 Bad Request`.
</Warning>

## Category schema

All five category types share the same response shape:

<ResponseField name="id" type="integer">
  Unique identifier for the category.
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the category. Between 1 and 100 characters.
</ResponseField>

<ResponseField name="is_custom" type="boolean">
  `true` if this category was created by a user; `false` if it is a system default.
</ResponseField>

<ResponseField name="created_at" type="string (datetime)">
  ISO 8601 timestamp of when the record was created.
</ResponseField>

## Body Areas

Classify exercises by the region of the body being targeted (e.g. Cervical, Lumbar, Knee, Shoulder).

### List body areas

<Note>**GET** `/api/v1/categories/body-areas`</Note>

<CodeGroup>
```bash Request
curl http://localhost:8000/api/v1/categories/body-areas
```
</CodeGroup>

```json Response 200
[
  { "id": 1, "name": "Cervical", "is_custom": false, "created_at": "2026-01-01T00:00:00Z" },
  { "id": 2, "name": "Knee",     "is_custom": false, "created_at": "2026-01-01T00:00:00Z" },
  { "id": 3, "name": "Shoulder", "is_custom": false, "created_at": "2026-01-01T00:00:00Z" }
]
```

### Create body area

<Note>**POST** `/api/v1/categories/body-areas`</Note>

New categories are always created with `is_custom: true`.

<ParamField body="name" type="string" required>
  Name of the body area. Between 1 and 100 characters.
</ParamField>

<CodeGroup>
```bash Request
curl -X POST http://localhost:8000/api/v1/categories/body-areas \
  -H "Content-Type: application/json" \
  -d '{ "name": "Wrist" }'
```
</CodeGroup>

```json Response 201
{ "id": 10, "name": "Wrist", "is_custom": true, "created_at": "2026-02-10T11:00:00Z" }
```

### Update body area

<Note>**PUT** `/api/v1/categories/body-areas/{category_id}`</Note>

<ParamField path="category_id" type="integer" required>
  ID of the body area to update.
</ParamField>

<ParamField body="name" type="string" required>
  New name for the body area.
</ParamField>

<CodeGroup>
```bash Request
curl -X PUT http://localhost:8000/api/v1/categories/body-areas/10 \
  -H "Content-Type: application/json" \
  -d '{ "name": "Wrist/Hand" }'
```
</CodeGroup>

```json Response 200
{ "id": 10, "name": "Wrist/Hand", "is_custom": true, "created_at": "2026-02-10T11:00:00Z" }
```

### Delete body area

<Note>**DELETE** `/api/v1/categories/body-areas/{category_id}`</Note>

<ParamField path="category_id" type="integer" required>
  ID of the body area to delete. Must be a custom category (`is_custom: true`).
</ParamField>

<CodeGroup>
```bash Request
curl -X DELETE http://localhost:8000/api/v1/categories/body-areas/10
```
</CodeGroup>

```json Response 200
{ "message": "Body area deleted successfully" }
```
