## What is Digi Rehab?

Digi Rehab is a full-stack rehabilitation exercise prescription platform built on **FastAPI** and **React**. It gives healthcare providers a structured way to build and manage exercise programs for their patients — from cataloguing therapeutic exercises to issuing personalized prescriptions.

The system is organized around four core entities:

- **Joints** — body joints that can be affected (knee, shoulder, lumbar, etc.)
- **Problems** — specific conditions associated with a joint (pain, stiffness, post-surgical)
- **Exercises** — therapeutic exercises with instructions, media, difficulty levels, and multi-dimensional categories
- **Prescriptions** — patient-specific exercise plans with custom repetitions, sets, and instructions

## Architecture overview

```
Digi Rehab
├── Backend (FastAPI + SQLAlchemy)
│   ├── routers/        — API endpoint definitions
│   ├── services/       — Business logic layer
│   ├── schemas/        — Pydantic request/response models
│   └── database/       — SQLAlchemy models and connection
└── Frontend (React + TypeScript + Vite)
    ├── pages/          — Dashboard, Joints, Problems, Exercises, Prescriptions
    ├── components/     — Shared UI components
    └── api/            — Axios API client
```

## Get started

<Steps>
  <Step title="Install dependencies">
    Install Python packages and initialize the database.

    ```bash
    pip install -r requirements.txt
    python init_database.py
    ```
  </Step>
  <Step title="Start the API server">
    Run the FastAPI backend on port 8000.

    ```bash
    python main.py
    ```
  </Step>
  <Step title="Start the frontend">
    Launch the React development server.

    ```bash
    cd frontend && npm install && npm run dev
    ```
  </Step>
  <Step title="Open the app">
    Visit `http://localhost:5173` for the dashboard, or `http://localhost:8000/docs` for the interactive API explorer.
  </Step>
</Steps>

  See the [Quickstart](/quickstart) guide for a complete walkthrough including database configuration and your first prescription.
</Note>
